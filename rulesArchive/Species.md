# Species

Each `species` has a number of `traits` they can choose from at certain `levels`. All playable `species` are collectively referred to as "humans".

# Mensa

An ape-like human

# Dwarf

A mole-like human

# Elf

A plant-like human

# Gnome

A mouse-like human

# Fiendling

A demon-like human

# Eleming

A humanoid fused with an element

# Adlet

A dog-like humanoid

# Garuda

A bird-like humanoid

# Kappa

A turtle-like humanoid

# Fairy

An insectoid humanoid

# Karnabo

An elephant-like humanoid

# Minotaur

A cow-like humanoid

# Rajuqati

A cat-like humanoid

# Grroak

A frog-like humanoid
