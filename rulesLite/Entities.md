# Entity

Every physical thing in the game is an `entity`. Concepts, mechanics, and non-diegetic game elements are not `entities`.

# Creature

Any living, sentient `entity` is a `creature`; e.g. people, mobs, and animals.

# Object

Any non-living or non-sentient `entity` is an `object`; e.g. a computer, plants, rocks, and fungi.

# Non-Player Characters

A `non-player character` is a `creature` controlled by the GM.

# Player Character

A `player character` (or `PC`) is a `creature` controlled by one of the players.
