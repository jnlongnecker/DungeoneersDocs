# Tag

A `tag` is a marker that explains a lot about a feature (`item`, `action`, `activity`, etc) in as few words as possible. They may grant functionality, pose restrictions, or simply categorize the feature they mark.

# Long

Applies to: `Activities`

Takes a considerable amount of time to complete; roughly 3 hours.

# Average

Applies to: `Activities`

Takes a relatively long amount of time to complete; roughly 1 hour.

# Short

Applies to: `Activities`

Takes a relatively short time to complete; roughly 10 minutes.

# Ritual

Applies to: `Activities`

Requires special circumstances and/or `items` in order to complete, and may have an effect within a `range` or for certain `creatures`.

# Attack

Applies to: `Actions`

Category. Anything that refers to an `attack` refers to an `action` with this `tag`.

# Movement

Applies to: `Actions`

Category. Anything that allows a `creature` to `move`.

# Interact

Applies to: `Actions`

Category. Anything that interacts with things in the world in a way not inherently hostile.

# Quick

Applies to: `Actions`

Grants +1 `priority` to the `action`.

# Slow

Applies to: `Actions`

Grants -1 `priority` to the `action`.

# Inherit

Applies to: `Actions`

Special; adopts the `tags` of the `action` referenced in addition to the existing `tags`.

# Passive

Applies to: `Traits`, `Feats`, `Skills`, `Blessings`

Does not use an `action` during a `procedure` and is always active.

# Boon

Applies to: `Traits`

Category. Any `trait` that is beneficial.

# Bane

Applies to: `Traits`

Category. Any `trait` that is harmful.
